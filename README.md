# Python智能健康检测系统

## 核心功能
- 登录注册
- 健康知识
- 健康资讯
- 用户管理
- 健康数据管理
- 饮食建议
- 运动计划
- 健康评估
- 睡眠指导
- 健康检测

## 适用场景
- 健康管理机构
- 健康教育平台
- 个人健康管理

## 技术特点
- 使用Django作为后端框架，提供稳定的服务
- 利用MySQL进行数据存储，确保数据安全可靠
- 前端采用Vue，提升用户体验
